# psj-lang (박상진 언어)

`umjunsik-lang` 문법을 오마주한 Rust 구현체입니다.

## 키워드 치환

- `엄` → `박`
- `준` → `상`
- `식` → `진`
- `동탄` → `게이`
- `어떻게` → `중국인`
- `이 사람이름이냐ㅋㅋ` → `게이 박상진`

## 실행

```bash
cargo run -- examples/hello.psj
```

입력(예: `진?`)이 필요하면 표준입력으로 공백 구분 정수를 넣으면 됩니다.

```bash
echo "3 4" | cargo run -- examples/input.psj
```

## 문법 요약

- 프로그램 시작: `중국인`
- 프로그램 끝: `게이 박상진`
- 변수 대입: `박...상{식}` (`박` 개수 = 변수 인덱스)
- 변수 참조: `박...`
- 정수: `.`(+1), `,`(-1), 공백(곱셈)
- 입력: `진?`
- 정수 출력: `진{식}!`
- 문자 출력: `진{식}ㅋ` (없으면 개행)
- 조건 실행: `게이{식}?{명령}` (식이 0이면 실행)
- 점프: `상{식}`
- 종료: `화이팅!{식}`

## 예시

`examples/hello.psj`

```txt
중국인
진........... ........ㅋ
게이 박상진
```
